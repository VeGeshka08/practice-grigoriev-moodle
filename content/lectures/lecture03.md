# Лекция 3 — Фишинг и социальная инженерия: Как не стать жертвой мошенников

Тема: Методы обмана в интернете и способы защиты от них.

Цель урока: Научиться распознавать фишинговые сайты и письма, понимать принципы социальной инженерии.

Время: ~20 мин.

Ключевые понятия: фишинг, социальная инженерия, фишинговые сайты, претекстинг, квипро-кво, фишинговые письма.

---

## 1) Введение

Фишинг (от англ. fishing — рыбная ловля) — это вид интернет-мошенничества, целью которого является получение доступа к конфиденциальным данным пользователей: логинам, паролям, данным банковских карт. Мошенники "забрасывают удочку" в виде поддельных писем, сайтов или сообщений и ждут, пока жертва "клюнет".

Социальная инженерия — это метод психологического манипулирования людьми с целью получения конфиденциальной информации. Злоумышленники не взламывают технически сложные системы, а "взламывают" человека.

## 2) Основная часть

### 2.1 Типы фишинговых атак

| Тип атаки | Описание | Пример |
|-----------|----------|--------|
| Email-фишинг | Массовая рассылка писем, маскирующихся под известные компании | Письмо "от банка" с требованием срочно подтвердить данные |
| Спеар-фишинг | Целевая атака на конкретного человека или организацию | Письмо, адресованное лично сотруднику, с поддельными данными |
| Вишинг | Голосовой фишинг через телефонные звонки | Звонок "из службы безопасности банка" с просьбой назвать код из СМС |
| Смишинг | Фишинг через SMS-сообщения | SMS "Ваша посылка задержана, перейдите по ссылке" |
| Клон-фишинг | Создание точной копии легитимного сайта | Сайт, визуально идентичный сайту банка, но с другим адресом |

### 2.2 Методы социальной инженерии

1. Претекстинг — создание вымышленного сценария для получения данных.  
   *Пример:* Звонок от "сотрудника техподдержки", который "помогает" решить проблему, но на самом деле запрашивает пароль.

2. Квипро-кво (Quid pro quo) — предложение услуги в обмен на информацию.  
   *Пример:* "Я из отдела IT, помогу вам настроить программу, но для этого нужно сказать ваш пароль".

3. Приманка (Baiting) — обещание выгоды для получения данных.  
   *Пример:* Бесплатная флешка, содержащая вредоносное ПО, оставленная в офисе.

4. Тейлигейтинг (Tailgating) — физическое проникновение вслед за сотрудником.  
   *Пример:* Мошенник проходит в офис, следуя за сотрудником через турникет.

## 3) Примеры для распознавания

### Признаки фишингового письма:

1. Срочность и угроза:  
   ✗ "Ваш аккаунт будет заблокирован через 24 часа!"  
   ✓ Банки и сервисы никогда не требуют срочных действий через ссылки в письмах

2. Ошибки в адресе отправителя:  
   ✗ support@paypa1.com (цифра 1 вместо буквы l)  
   ✗ bank@mail-ru-support.ru  
   ✓ Официальный адрес должен совпадать с доменом компании

3. Грамматические ошибки и странный перевод:  
   ✗ "Уважаемый клиент, ваш аккаунт был скомпрометирован"  
   ✗ "Пожалуйста, перейти по ссылке для верификации"

4. Подозрительные ссылки:  
   ✗ http://bank-verification.ru/login  
   ✓ Перед переходом наведите курсор мыши на ссылку — всплывет реальный адрес
